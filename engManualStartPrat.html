EEC STARTUP PROTECTIONS DISABLED DURING MANUAL STARTING MODE – MONITOR ALL PARAMETERS AND ABORT START IF ANY OF THE FOLLOWING OCCUR:

    ABORT START IF ANY OF THE FOLLOWING OCCUR

  • The oil pressure does not increase
  • No N1 indication when the N1 is greater than 48%
  • Ignition System and/or Fuel Flow does not come online
  • No increase in EGT 20 seconds after Fuel Flow increase
  • More than 90 seconds after fuel flow to achieve Idle N2
  • Fuel or Ignition is accidently stopped
  • Loss of EGT indication / EGT approaches startup limit
  • Sluggish N2 acceleration with a fast increase in EGT
  • Abnormal vibrations detected / Engine surge observed

    • ABORTED START MUST BE FOLLOWED UP BY A DRY-MOTOR FOR 30 SECONDS OR UNTIL EGT IS LESS THAN 250° C

  • ENGINE MODE SELECTOR.......IGN START
  • AIR PRESSURE AT START VALVE.......BETWEEN 30 & 40 PSI
  • ENGINE MANAUAL START BUTTON 1(2).......ON
  • CHRONO.......START WHEN VALVE OPENS
  • DRY CRANK.......30 SECONDS
  • COOLING TIME (IF REQUIRED).......MONITOR
  • OIL PRESSURE.......OBSERVE AN INCREASE
  • N2 INDICATION.......MAX MOTORING (MIN 18%)
  • ENGINE MASTER SWITCH 1(2).......ON
  • FUEL FLOW.......OBSERVE AN INCREASE
  • ENGINE 1(2) IGNITER A OR B.......ON
  • EGT RISE.......OBSERVE AN INCREASE (MAX 1084°)
  • IGNITION / STARTER CUT-OUT.......OBSERVE AT 55% N2
  • ENGINE MANAUAL START BUTTON 1(2).......OFF

    AT IDLE CHECK FOR

  • N1 = 19%
  • EGT = 440°C
  • N2 = 59%
  • FF = 600 LBS / HR

NOTE: THESE VALUES ARE APPROXIMATE AND CAN VARY DUE TO AGE OF ENGINE
